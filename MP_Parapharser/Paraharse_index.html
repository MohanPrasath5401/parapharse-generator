Humanized Paraphraser

Local Python tool that rewrites text to sound more natural and avoid AI detectors

How to use (recommended – full power)

  1. Clone this repository:
    git clone https://github.com/YOUR-USERNAME/paraphraser-humanized.git
  2. Install dependencies:
    pip install torch transformers (add --extra-index-url https://download.pytorch.org/whl/cu121 if you want GPU)
  3. Run:
    python paraphraser.py
  4. Paste your text → get 5–8 natural-sounding variants

Best results: always lightly edit the output in your own words.
